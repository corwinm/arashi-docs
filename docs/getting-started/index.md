Use this section when you are new to Arashi and need a quick setup and first workflow.

## Install

Install from npm:

```bash
npm install -g arashi
```

## First Workflow

```bash
arashi init
arashi add git@github.com:your-org/frontend.git
arashi create feature-docs-bootstrap
arashi status
```
