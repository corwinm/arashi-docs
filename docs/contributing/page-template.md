Copy this template when creating a new page.

```markdown
---
title: <Page Title>
description: <One-sentence purpose>
---

## Overview

Explain the goal and who should use this page.

## Steps

1. Step one.
2. Step two.
3. Step three.

## Validation

- Mention required checks or prerequisites.

## Related

- [Related Page](/path/)
```
